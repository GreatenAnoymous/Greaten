Projects

  • [1]Polynomial Time Multi-Robot Path Planning (MRPP) on well-connected Grids with Near-optimal Expected Makespan Demo

    • Accepted by RSS2022: https://arxiv.org/abs/2201.08976
    • [Motivation] Solving MRPP in polynomial time with makespan optimality guarantee or grids and sorting center warehouses.
    • [Contributions]
      • (i). Proposed a polynomial time multi-robot path planning method that can solve large and dense instances on well-connected grids.
      • (ii). Proved that this algorithm returns asymptotic 1.5 expected makespan-optimal solution on square grids for random MRPP instances.
      • (iii). The algorithm is highly scalable, which can solve instances with up to 30000 robots on 300x300 grid map. There does not exist a method before with such scalibility and optimaltiy guarantees.This work is submitted to RSS-2022

    • [Motivation] Study divide-and-conquer based heuristics for MRPP
    • [Contributions]

      • Proposed space split heuristic to improve the scalibility without sacrificing the makespan optimality
